## 🎯 Learning Outcomes

- **Understanding Backend Authorization**  
  Learned the fundamentals of authentication and authorization and how backend servers control access to protected resources.

- **Building a Python Backend Server**  
  Gained experience in creating a backend server using Python to manage user requests and secure application endpoints.

- **Implementing Token-Based Security**  
  Learned how to use token-based authentication (e.g., JWT) to verify user identity and maintain secure sessions.

- **Securing API Endpoints**  
  Developed the ability to protect API routes so that only authenticated and authorized users can access specific data or functionality.

- **Frontend–Backend Authorization Integration**  
  Understood how frontend applications communicate with the backend server to perform secure login, authorization checks, and access control.
